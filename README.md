## Set up

1. Setup Vundle

    ```
      $ git clone https://github.com/gmarik/vundle.git ~/.vim/bundle/vundle
    ```

2.  Create Symlinks:

    `$ rm ~/bashrc && rm ~/.vimrc`  
    `$ cd .dotfiles`  
    `$ ln .bashrc ~/.bashrc && ln .vimrc ~/.vimrc`  

3. Install configured bundles:  
    Launch `vim`, run `:BundleInstall`
